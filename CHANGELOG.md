# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-XX-XX

### Added
- Initial release of Talker
- Natural language voice commands for window control
- AI-powered intent parsing using local LLM
- Support for multiple speech-to-text engines (macOS native, Whisper, Sphinx)
- Fuzzy app name matching
- Context-aware command processing
- Preset window layouts for multi-monitor setups
- Hotkey support for triggering voice commands
- Chrome tab switching support
- App focusing, placement, and closing commands
- Multi-command support (parsing multiple commands in one utterance)
- Clarification dialogs for ambiguous commands

### Features
- Voice-controlled window management on macOS
- Support for multiple monitors (main, left, right)
- Window maximization support
- Running and installed app detection
- Offline speech recognition (no cloud services required)
